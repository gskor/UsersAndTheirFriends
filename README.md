# UsersAndTheirFriends

Данное приложение выводит список пользователей с их текущим статусом (условно онлайн либо оффлайн: active/inactive), при тапу можно посмотреть подробную информацию о пользователе и список его друзей.

Приоржение использует URLSession для загрузки JSON-данных из интернета, преобразовывает их в типы Swift с помощью Codable, а затем отображает их пользователю с помощью NavigationStack, List и других инструментов.

URL-адрес, который используется для загрузка образца данных, выглядит следующим образом: https://www.hackingwithswift.com/samples/friendface.json — это огромная коллекция случайно сгенерированных данных, в нашем случае, пользователей.

Как видно, это массив людей, у каждого из которых есть идентификатор, имя, возраст, адрес электронной почты и другие данные. У них также есть массив строк тегов и массив друзей, у каждого из которых есть имя и идентификатор.
