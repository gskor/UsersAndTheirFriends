# UsersAndTheirFriends

Мобильное приложение на **SwiftUI**.  
UsersAndTheirFriends — это приложение-конвертер единиц измерения длины: пользователь выбирает исходную и итоговую меру длины, вводит значение и получает результат конвертации.

В проекте используются следующие инструменты для работы с UI:
Picker, segmented control, Form
